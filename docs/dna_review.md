## Ⅲ. 적합도 산출 공식 (계획서 Ⅷ장 재정리)

```
학과 적합도(%) = ( Σ(학생 매칭축 점수 × 학과 DNA 가중치) ÷ 학과 최대점수 ) × 100

  - 학생 매칭축 점수: 0~5점 (8개 진단축 응답을 24개 매칭축으로 환산)
  - 학과 DNA 가중치: 0~5점 (본 표)
  - 학과 최대점수 = Σ(학과 DNA 가중치) × 5

31개 학과에 대해 동시 계산 → 상위 5개를 TOP5 추천, 6~8위를 비교탐색 학과로 제시
```

## Ⅳ. 후속 작업 (학과장 검토 단계에서 확인 필요)

1. **뷰티매니지먼트과 메이크업&네일전공**: 계획서는 '4개 전공'으로 명시되어 있으나 자료 ZIP에는 3개 전공만 포함되어 있음. → 메이크업&네일전공 가이드북 추가 수령 후 별도 행 추가 필요 (예상 코드: `ART_BTY_MN`).
2. **드론정보공학과(SMART_DRONE)**: 2024학년도 가이드북만 제공됨. 2025학년도 개정본 입수 시 재검토.
3. **학과장 보정 (계획서 Ⅶ-3 (2)단계)**: 각 학과에 본 DNA 초안을 보내 0~5점 가중치 검토·회신 받기. 회신 결과로 1차 표준 DNA표 확정.
4. **계열 분류 재검토**: AI.SW융합학부의 전자공학과는 HW축에 5점 부여했으나 명칭상 'AI.SW' 학부에 속하므로 학과장 의견 추가 필요.
5. **검토 우선순위 학과**: 가중치 합계가 낮은(=정체성이 좁은) 학과는 변별력이 떨어질 수 있으므로 우선 검토 — 세무회계과(11.0), 항공서비스과(11.0), 일본어학과(12.5), 중국어학과(12.5).
